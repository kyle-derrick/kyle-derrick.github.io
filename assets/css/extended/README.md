# 博客美化样式说明

这组 CSS 文件为 Hugo PaperMod 主题提供了全面的视觉美化，采用简约现代风格，注重细节和移动端适配。

## 文件说明

| 文件 | 功能 |
|------|------|
| `custom-vars.css` | 统一的 CSS 变量系统（配色、阴影、圆角、间距、动画） |
| `custom-toc.css` | 右侧悬浮目录样式 - 毛玻璃效果、圆角卡片、滚动高亮 |
| `custom-post-toc.css` | 左侧文章目录样式 - 树形结构、优雅边框 |
| `custom-profile.css` | 个人主页样式 - 标题、按钮、社交图标 |
| `custom-post.css` | 文章阅读样式 - 排版、代码块、引用、表格 |
| `custom-entries.css` | 文章列表卡片 - 悬停动画、标签、分页 |
| `custom-nav.css` | 导航栏和页脚 - 毛玻璃头部、移动菜单 |
| `custom-global.css` | 全局样式 - 滚动条、选择文本、基础元素 |

## 主要特性

### 1. 配色系统
- 主色调：`#2563eb` (蓝色)
- 深浅主题自动适配
- 语义化颜色命名

### 2. 阴影系统
- `--shadow-sm`:  subtle 微阴影
- `--shadow`:  普通阴影
- `--shadow-md`: 中等阴影
- `--shadow-lg`: 强阴影

### 3. 圆角系统
- 从 `6px` 到 `9999px` (全圆角) 的完整层级

### 4. 目录优化
- 毛玻璃背景效果 (backdrop-filter)
- 优雅的圆角边框
- 当前项高亮指示条
- 移动端自适应布局

### 5. 移动端适配
- 响应式断点：1279px, 768px, 480px
- 移动端目录改为文档流布局
- 导航栏固定顶部
- 触控友好的按钮尺寸

## 使用方法

这些样式会自动被 Hugo 加载，无需额外配置。要修改样式，直接编辑对应的 CSS 文件即可。

## 注意事项

1. 修改 `custom-vars.css` 中的变量可以全局调整配色和尺寸
2. 所有过渡动画时间都在变量中统一定义
3. 深色模式通过 `.dark` 类自动适配
